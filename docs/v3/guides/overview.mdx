## Integration Categories

### **Instant Setup** (< 5 minutes)
- **MCP Integration**: Single prompt setup in Claude Code
- **Claude Code Plugin**: Native integration with zero configuration

### **Agent Frameworks** (10-30 minutes)
- **LangGraph**: Persistent memory layer for multi-agent systems
- **CrewAI**: Cross-mission agent memory and coordination
- **OpenClaw**: Multi-channel deployment with unified memory
- **Hermes Agent**: Terminal agent with advanced tool capabilities
- **Agent Zero**: Plugin-based modular agent architecture

### **Platform Bots** (30-60 minutes)
- **Discord/Telegram**: Chat bots with user memory and context
- **n8n**: No-code workflow integration and automation
- **Reachy Mini**: Embodied robotics with voice and memory

### **Data Migration** (1-2 hours)
- **Gmail Import**: Bulk email history ingestion
- **Granola Import**: Meeting transcript and participant mapping
- **Mem0 Migration**: System migration from existing memory backends

## Next Steps

1. **Start with [Quickstart](/v3/documentation/introduction/quickstart)** to set up your Honcho workspace
2. **Choose an integration** based on your existing stack or target platform
3. **Follow the specific guide** for step-by-step implementation
4. **Explore advanced features** in the [core concepts](/v3/documentation/core-concepts/architecture) documentation
5. **Join the community** for support and to share your implementations

Need help choosing? The **MCP Integration** is the fastest way to see Honcho in action, while **LangGraph** or **CrewAI** are ideal starting points for building production AI systems.
