# Features

Honcho provides a comprehensive set of features for building AI applications with persistent user memory and context management.

## Core Features

### User Context Management
- **Per-User Isolation**: Each user has their own data space
- **Session-Based Organization**: Conversations organized by sessions
- **Cross-Session Memory**: Access user context across multiple conversations
- **Metadata Support**: Rich metadata on all entities

### Vector Storage & Search
- **Automatic Embeddings**: Documents are automatically embedded
- **Semantic Search**: Find relevant content by meaning, not keywords
- **Collection Management**: Organize documents in user-scoped collections
- **Similarity Scoring**: Cosine similarity for relevance ranking

### Message & Memory Management
- **Message History**: Complete conversation logging
- **Metamessages**: Derived insights and analysis
- **Fact Extraction**: Automatic fact derivation from conversations
- **Context Synthesis**: Intelligent context assembly for AI responses

## Advanced Features

### Peer Paradigm
A conversation model where the AI acts as a peer rather than a traditional assistant:
- **Equal Participant**: AI participates as a conversation partner
- **Shared Context**: Both parties contribute to shared understanding
- **Collaborative Discovery**: Joint exploration of topics and ideas
- **Natural Interaction**: More human-like conversation patterns

### Queue Status
Real-time monitoring of background processing:
- **Async Operations**: Non-blocking processing for performance
- **Status Tracking**: Monitor job progress and completion
- **Error Handling**: Graceful failure recovery
- **Batch Processing**: Efficient handling of multiple operations

### Ingest Arbitrary Data
Flexible data ingestion capabilities:
- **Multiple Formats**: Text, documents, structured data
- **Batch Import**: Bulk data loading
- **Custom Processing**: Configurable data transformation
- **Metadata Preservation**: Maintain data provenance and context

### Advanced Search
Beyond basic vector search:
- **Hybrid Search**: Combine vector and keyword search
- **Filtered Search**: Search within specific metadata criteria
- **Multi-Collection Search**: Search across multiple collections
- **Temporal Search**: Find content from specific time periods

### Scoped API Keys
Fine-grained access control:
- **App-Level Keys**: Access to specific applications only
- **User-Level Keys**: Access to specific user data
- **Resource-Level Keys**: Access to specific collections or sessions
- **Time-Limited Keys**: Temporary access tokens
- **Permission-Based Keys**: Read-only, write-only, or full access
