📖 Integration Guide

To integrate Honcho memory with your Hermes agent:

# Python example
import requests

# Create workspace
workspace = requests.post('http://localhost:8000/v1/workspaces', 
    json={'name': 'klimashift'}).json()

# Create peer (user/agent)
peer = requests.post(f'http://localhost:8000/v1/workspaces/{workspace["id"]}/peers',
    json={'name': 'user_123'}).json()

# Create session
session = requests.post(f'http://localhost:8000/v1/workspaces/{workspace["id"]}/peers/{peer["id"]}/sessions',
    json={'name': 'conversation_1'}).json()

# Add message with memory
message = requests.post(f'http://localhost:8000/v1/workspaces/{workspace["id"]}/sessions/{session["id"]}/messages',
    json={'role': 'user', 'content': 'Hello!'}).json()
